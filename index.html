r2r-designer

An Editor for mapping relational Databases to RDF.

View the Project on GitHub rdanitz/r2r-designer

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

This version is an early prototype. Do not use in production.

Requirements

You will need:

  • node.js 20.10.26 or above,
  • Bower 1.3.1 or above,
  • Leiningen 1.7.0 or above.
  • Grunt 0.4.4 or above.

Installation

  npm install
  bower install

Usage

To start the web application for the R2R designer, run:

    grunt serve

To start the web service for the R2R designer, run:

    lein run
